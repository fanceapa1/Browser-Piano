How to use

Play a song on the browser piano by clicking/tapping on the keys, or by pressing the corresponding key on your keyboard.

The piano is split into two octaves, for accesibility purposes. A dropdown menu is available below the keys, where you can select between different octaves, depending on the song you want to play.

If you want to change visual elements of the piano, such as the color or the displaying of the notes/keybinds on the piano's keys, you can do that in the settings tab below this section.

How it works

Each of the keys is mapped to a unique audio file. Whenever you press a key, the corresponding audio file is played by the website.

When you switch the octave, the set of audio files is changed to match the selected octave. You can even have both of the sets of keys set to the same octave, try to create some unique harmonies!
